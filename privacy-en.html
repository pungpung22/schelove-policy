Privacy Policy

Schelove - K-pop Schedule Calendar

1. Purpose of Collecting and Using Personal Information

Schelove (the "App"), operated by PungPung Studio (the "Operator"), collects and uses personal information for the following purposes:

  • Member registration and management: Identity verification and user identification for membership services
  • Service provision: Schedule registration, D-day notifications, AI speech style system
  • Service improvement: Analysis of service usage records for service enhancement
  • Notification services: Push notifications for registered schedules

2. Personal Information Collected

Required Items

  • Google account information: Email address, name, profile photo URL
  • Device information: Device identifier (for push notifications)

Information Generated During Service Use

  • Registered schedule information (event name, date, category)
  • Preferred idol/group information
  • AI speech style settings
  • D-day notification settings
  • Service usage records

3. Retention and Use Period of Personal Information

User's personal information is, in principle, destroyed without delay when the purpose of collection and use is achieved.

  • Upon account deletion: Immediately deleted
  • Service usage records: Immediately deleted upon account deletion

4. Provision of Personal Information to Third Parties

The App does not, in principle, provide users' personal information to external parties. However, the following cases are exceptions:

  • When the user has given prior consent
  • When required by law or requested by investigative authorities in accordance with legal procedures

5. Procedure and Method of Personal Information Destruction

User's personal information is transferred to a separate DB after the purpose is achieved and is destroyed after a certain period of storage according to internal policies and information protection reasons under relevant laws.

Destruction Method

  • Personal information stored in electronic file format is deleted using technical methods that prevent record reproduction.

6. User Rights and How to Exercise Them

Users can exercise the following rights at any time:

  • Request to view personal information
  • Request for correction of errors
  • Request for deletion
  • Request to stop processing

These rights can be exercised through the settings menu in the App or via email (pung0805@gmail.com).

7. How to Delete Your Account

Account deletion can be done as follows:

  • In the App: Settings > Delete Account button
  • Email: Request deletion at pung0805@gmail.com

The following data will be permanently deleted immediately upon account deletion:

  • Member information (email, name, profile photo)
  • All registered schedules
  • Preferred idol/group information
  • AI speech style settings
  • D-day notification settings
  • All service usage records

Deleted data cannot be recovered. Please decide carefully.
